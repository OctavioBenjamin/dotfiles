## 🎨 The Stack

Este repositorio contiene mis archivos de configuración para:

*   **OS:** [Arch Linux](https://archlinux.org/) 🐧
*   **Window Manager:** [Hyprland](https://hyprland.org/) (Wayland)
*   **Bar:** [HyprPanel](https://hyprpanel.com/) (Altamente personalizable)
*   **Terminal:** [Ghostty](https://ghostty.org/) (GPU accelerated)
*   **Shell:** [Zsh](https://www.zsh.org/) (+ Oh My Zsh) & [Fish](https://fishshell.com/)
*   **Editor:** [Neovim](https://neovim.io/) (LazyVim configuration)
*   **Theming:** [Matugen](https://github.com/InSync-Design/matugen) (Material You color generation)
*   **File Manager:** [Thunar](https://docs.xfce.org/xfce/thunar/start)
*   **App Launcher:** [Wofi](https://hg.sr.ht/~scoopta/wofi)
*   **Wallpaper:** [swww](https://github.com/L_S_W_W/swww)
*   **System Info:** [Fastfetch](https://github.com/fastfetch-cli/fastfetch) (Custom pinguino config)

## ✨ Características

*   **Dynamic Theming:** Los colores del sistema cambian según el fondo de pantalla gracias a Matugen.
*   **Dual Shell:** Zsh configurado con Powerlevel10k para uso diario y Fish para una experiencia interactiva rápida.
*   **Neovim IDE:** Configuración completa basada en LazyVim con soporte para múltiples lenguajes.
*   **HyprPanel:** Una barra de estado moderna con widgets integrados para control de medios, redes y más.

## 🚀 Instalación

> [!WARNING]
> No ejecutes una instalación a ciegas. Revisa los archivos y adapta las rutas a tu usuario.

### 1. Clona el repositorio
```bash
git clone https://github.com/octaviobenjamin/dotfiles.git
cd dotfiles
```

### 2. Copia las configuraciones
Actualmente, puedes copiar las carpetas a tu `~/.config`:

```bash
cp -r .config/* ~/.config/
cp .zshrc ~/
```
